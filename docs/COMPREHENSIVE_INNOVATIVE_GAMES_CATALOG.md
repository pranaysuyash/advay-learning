## ğŸ› ï¸ TECHNICAL REQUIREMENTS

### For AR Games (External Camera)

**Recommended Setup:**
```
Level 1: Laptop + Built-in camera (single camera fallback)
Level 2: Laptop + USB webcam (external top-down) 
Level 3: Laptop + Document camera (premium experience)
Level 4: Tablet + Stand (portable AR)
```

**Browser APIs:**
- `getUserMedia()` for camera access
- `enumerateDevices()` for multi-camera
- Canvas API for AR overlay
- WebGL for 3D objects (Three.js)
- Web Audio API for spatial sound

**MediaPipe Models:**
- Hand Landmarker (for interaction)
- Face Landmarker (for expression/attention)
- Pose Landmarker (for body movement)
- Image Segmenter (for background removal - optional)

### For Science Simulations

**Physics Engine:** Matter.js or Cannon.js
**Chemistry:** Custom state machines for reactions
**Graphics:** HTML5 Canvas or WebGL
**Audio:** Web Audio API for experiment sounds
