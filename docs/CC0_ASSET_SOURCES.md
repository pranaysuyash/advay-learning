## ⚠️ Important Notes

### License Verification
- **Always verify CC0 status** before using
- Some sites mix licenses (CC0, CC-BY, proprietary)
- Check for attribution requirements

### File Formats
| Format | Best For | Web Ready |
|--------|----------|-----------|
| **GLB/glTF** | 3D models | ✅ Yes |
| **PNG** | 2D sprites, UI | ✅ Yes |
| **SVG** | Vector graphics | ✅ Yes |
| **OBJ** | 3D models | ⚠️ Needs conversion |
| **FBX** | Animated models | ⚠️ Needs conversion |
| **MP3/OGG** | Audio | ✅ Yes |

### Conversion Tools
- **Blender** (free) - Convert FBX/OBJ to glTF
- **TexturePacker** - Create spritesheets
- **SVGO** - Optimize SVG files
