### PHONICS GAMES (New!)

#### 9. Phonics Sounds (Pip Spells It!) - P0 ⭐

**Age:** 3-6 | **Effort:** 1.5 weeks

**Concept:** Pip makes a letter SOUND, child traces the letter.

```
┌─────────────────────────────────────┐
│     🔊 Pip Spells It!               │
│                                     │
│         🦉 "Buh! Buh!"               │
│        (Pip making B sound)         │
│                                     │
│     [ Trace the letter: B ]         │
│     ┌─────────────────────┐         │
│     │        B            │ ← ghost │
│     └─────────────────────┘         │
│                                     │
│     🔊 Repeat sound   ❓ Hint        │
└─────────────────────────────────────┘
```

**Mechanics:**

- Pip plays letter SOUND (not name)
- Ghost letter appears faintly
- Child traces the letter
- On complete: "Buh is for Ball! 🏐"

**Progression:**

1. Easy sounds (A, B, C, D, M, S)
2. Medium sounds (E, F, G, H, L, N)
3. Similar sounds (B/D, P/T)
4. Vowel variations (long/short)

**Assets Needed:**

- [ ] Phoneme audio recordings (26+ sounds)
- [ ] Word examples with images
- [ ] Pip "teaching" animations

#### 10. Phonics Tracing (Sound It Out!) - P1

**Age:** 4-6 | **Effort:** 1.5 weeks

**Concept:** Kid traces, Pip sounds it out IN REAL-TIME.

```
AS child traces S:
🔊 "Sssssssss..." (continuous sound)

AS child traces T:
🔊 "T!" (single burst)
```

**Sound Types:**

| Type       | Letters             | Behavior              |
| ---------- | ------------------- | --------------------- |
| Continuous | S, F, M, N, R, L, Z | Sound while tracing   |
| Burst      | B, D, P, T, K, G    | Sound at stroke start |
| Vowel      | A, E, I, O, U       | Held note             |

#### 11. Rhyme Time - P1

**Age:** 4-6 | **Effort:** 1 week

**Concept:** Match rhyming words.

```
Which rhymes with CAT? 🐱
[ DOG 🐕 ]  [ BAT 🦇 ]  [ CAR 🚗 ]
              ↑ correct
```

**Rhyme Families:**

- -at: cat, bat, hat, mat, rat, sat
- -an: can, fan, man, pan, van, ran
- -ig: big, dig, fig, pig, wig
- -op: cop, hop, mop, pop, top
- -ug: bug, hug, jug, mug, rug

## 🧱 KENNEY ASSET REUSE GAMES (Platformer Pack)

These game concepts heavily utilize the 2D characters, tiles, enemies, and collectibles from the Kenney Platformer Pack, maximizing asset reuse and providing a cohesive feel.

#### 1. Math Jumpers (Number Line Platformer)

**Age:** 4-7 | **Effort:** 1 week
**Concept:** A side-scrolling or static platformer where tiles are labeled with numbers. The game asks "What is 2 + 3?", and the player must move their character (e.g., the alien) to jump onto the tile labeled "5".
**Assets Used:** `characters`, `tiles` (for platforms), `sounds` (jump/land).

#### 2. Spelling Run (Word Collector)

**Age:** 5-8 | **Effort:** 1.5 weeks
**Concept:** A runner or maze game where letters are scattered across platforms. The child is prompted to spell a word (e.g., "CAT") and must collect the letters in the correct order while dodging passive enemies like slimes.
**Assets Used:** `characters`, `tiles`, `enemies` (as obstacles), `sounds`.

#### 3. Counting Collect-a-thon (Scavenger Hunt) - ✅ LIVE

**Age:** 3-5 | **Effort:** 1 week
**Concept:** A relaxed, non-scrolling environment built using the platform tiles. The prompt asks to "Collect exactly 4 blue gems!". The child guides the character to grab the correct items and avoid the wrong ones.
**Assets Used:** `characters`, `tiles` (level building), `collectibles` (gems, coins, keys, stars), `hud`.

#### 4. Catch & Sort (Falling Objects)

**Age:** 3-5 | **Effort:** 1 week
**Concept:** The player moves the character left/right along the bottom of the screen. Items fall from the sky. The game instructs: "Catch only the stars, avoid the enemies!" or "Catch all the red items!".
**Assets Used:** `characters`, `collectibles` (falling good items), `enemies` (falling bad items), `backgrounds`.

#### 5. Logic Box Push (Sokoban Style)

**Age:** 5-8 | **Effort:** 2 weeks
**Concept:** A puzzle game where the player must push crates onto specific switches to open a door or bridge a gap. Teaches spatial reasoning and logic planning.
**Assets Used:** `characters`, `tiles` (crates, bridges, solid ground, switches), `sounds`.

## 🚀 NEW GAME IDEAS (MediaPipe + Browser)

### BEAUTIFUL & CUSTOMIZABLE GAMES

#### 18. Shadow Portal (Segmentation) - P0 ⭐

**Age:** 3-6 | **Effort:** 2 weeks
**Concept:** Your silhouette "blocks" or "guides" falling light particles into portals.
**Control:** Move body, raise arms, make "tunnels" with silhouette.
**Why it works:** No precise clicking, visually beautiful, works even with lag.
**Customization:** Themes (space, jungle, neon), particle styles, portal shapes.

#### 19. Spell Painter (Hands) - P1

**Age:** 4-6 | **Effort:** 2 weeks
**Concept:** Draw a rune in the air; game recognizes stroke pattern; spell triggers physics.
**Control:** Index finger trail; dwell to "commit".
**Learning mode:** Shapes, letters, basic geometry.
**Pitfalls:** Recognition needs tolerance; keep rune set small (8–12) initially.

#### 20. Music Conductor (Pose + Hands + WebAudio) - P1

**Age:** 4-8 | **Effort:** 2.5 weeks
**Concept:** Conduct an orchestra. Faster hands = tempo; left/right hand height = volume of sections.
**Control:** Simple continuous mapping.
**Customization:** Instrument packs, visualizers, "lofi mode" vs "EDM mode".
**Why it's sticky:** Instant feedback, zero menus.

#### 21. Bubble Biology (Pinch/Grab) - P2

**Age:** 5-8 | **Effort:** 2 weeks
**Concept:** Pinch to "grab" cells, sort them, merge to evolve creatures.
**Learning:** Basic bio taxonomy, cell parts, "predator/prey" concepts.
**Open data:** Optional Wikipedia summaries for discovered species.

#### 22. Mirror Maze (Face + Head tilt) - P2

**Age:** 3-6 | **Effort:** 1.5 weeks
**Concept:** Steer a ball through a maze using head tilt; blink to drop a bridge.
**Why it works:** One-person, low arm fatigue, accessible.
**Pitfalls:** Head tilt sensitivity must be adjustable.

### LEARNING GAMES THAT DON'T FEEL LIKE HOMEWORK

#### 23. Letter Catcher (Hands / Zones) - P1

**Age:** 3-5 | **Effort:** 1.5 weeks
**Concept:** Letters float in; catch only "A" or only vowels; later phonics.
**Control:** Move hand to basket, dwell to "lock".
**Customization:** Language packs (EN/HI/Kannada), fonts, objects.

#### 24. Math Smash (Pose zones) - P1

**Age:** 4-7 | **Effort:** 1.5 weeks
**Concept:** Two answers appear left/right; step into the correct zone.
**Control:** Body position, not hands.
**Scales well:** Difficulty ramps, timed mode, co-op with two players.

#### 25. Circuit Builder (Hands) - P2

**Age:** 5-8 | **Effort:** 2 weeks
**Concept:** Drag components in mid-air to complete a circuit; current flows when correct.
**Learning:** Logic gates, series/parallel, cause-effect.
**Pitfalls:** Needs forgiving snapping and big targets.

#### 26. Weather Lab (Hands + Open-Meteo / NOAA style data) - P2

**Age:** 5-8 | **Effort:** 2.5 weeks
**Concept:** "Summon" weather by arranging pressure systems; then compare to real city weather.
**Exploration:** Pick Bangalore, Reykjavik, etc., see how patterns map to reality.
**Why it's cool:** Toys + real world.

#### 27. Language Puppet (Face + Hands) - P3

**Age:** 4-7 | **Effort:** 2 weeks
**Concept:** Your face drives a character; speak a word, character acts it out; kids repeat.
**Tech:** Web Speech API (optional) + face landmarks for expression.
**Pitfalls:** Speech on web varies by browser; keep it optional.

### EXPLORATORY "NASA AND FRIENDS" CONCEPTS

#### 28. NASA Sky Hunt (NASA APIs) - P2

**Age:** 5-9 | **Effort:** 3 weeks
**Concept:** You "lasso" constellations with your hand trail; unlock stories, missions, images.
**Data:** NASA APOD, Mars Rover photos, NeoWs asteroids, etc.
**Mechanic:** Gesture-based "connect the stars", then reveal real imagery.

#### 29. Planet Sandbox (Three.js + simple physics) - P2

**Age:** 6-9 | **Effort:** 3 weeks
**Concept:** Pinch to spawn moons, stretch to change orbit radius; learn gravity intuitively.
**Data tie-in:** Real planet parameters as presets (mass, radius).

#### 30. Earth Time Machine (NASA Earthdata / imagery sources) - P3

#### 31. ISS Docking Mini-game (orbital theme) - P2

**Age:** 6-9 | **Effort:** 2 weeks
**Concept:** Align docking using two-hand distance (throttle) + lateral hand movement (translation).
**Why it works:** Satisfying skill game, looks premium.

## 🎨 BEAUTIFUL AND CUSTOMIZABLE RENDERING

**If you want this to feel premium, don't ship "webcam with dots". Ship stylized output:**

- Use segmentation to turn the user into a clean silhouette (glow, ink, paper-cut).
- Render game in WebGL (Three.js) or Canvas2D with particles.
- Audio matters: Tone.js/WebAudio for responsive sound cues.
- Add a theme system: palette, particle style, typography, character pack, background pack.

---

## WHERE THESE IDEAS BREAK (AND HOW TO DESIGN AROUND IT)

- **Latency / jitter:** Avoid "tiny targets". Prefer zones, silhouettes, continuous mappings.
- **Kids pressing UI:** Go "hands-free UI": dwell-to-select, big side zones, voice optional.
- **Lighting variability:** Include a calibration step that is part of the story (e.g., "summon the portal: stand still for 2 seconds").
- **Device variability:** Degrade gracefully: 60fps visuals optional, core logic at 30fps.
- **Fatigue:** Long-arm games burn out. Mix face/head/pose modes.
