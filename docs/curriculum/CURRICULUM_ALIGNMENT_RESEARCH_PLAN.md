## Mapping Methodology

### Step 1: Extract NCERT Learning Outcomes

**For each NCERT outcome, identify:**
1. Age group
2. Domain
3. Specific skill
4. Observable behavior

**Example:**
```
NCERT Code: L-1.3
Age: 5-6 years
Domain: Language
Skill: Blends sounds to make words
Observable: Child can say "c-a-t = cat"
```

### Step 2: Map Games to Outcomes

**Template:**
```markdown
| Game | NCERT Code | Match Strength | Evidence |
|------|------------|----------------|----------|
| WordBuilder | L-1.3 | Strong | Direct blending practice |
```

### Step 3: Define Assessment Methods

**Pre-literate Assessment (no reading/writing required):**
- Accuracy rate
- Time to completion
- Error patterns
- Help requests
- Persistence (retries)
