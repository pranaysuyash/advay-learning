# Universe Vision Mapping Prompt v1.0

**Purpose:** A universal, project-agnostic prompt that maps ANY product/project to the complete universe of what it COULD become â€” finding blind spots, untapped potential, and the true scope of the vision.

**Works for:** SaaS, consumer apps, developer tools, hardware, marketplaces, content platforms, games, health tech, fintech, edtech, anything.
